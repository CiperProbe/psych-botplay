A Friday Night Funkin' custom botplay that forces psych engines botplay to always play in a song. Unlike the traditional botplay for Psych engine (That doesnt save score) This one can save score.

Currently this only has been tested for Manny Edition of Friday Night Funkin'

Too add this in your game, find the appropriate "script" folder for your Psych engines FNF mod and put it in

If you cant find it, then try to create it through: mods/yourfnfmodhere/scripts
